# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-01-XX

### Added
- Initial release
- User authentication (wallet + email/password)
- Role-based access (Student/Admin)
- Student profile management
- Admin dashboard for record uploads
- Wallet integration with APT transactions
- Move smart contract for badge NFTs
- Verifier registry system
- Event emission for on-chain actions
- Document upload/download functionality
- Social profile linking (GitHub, HackerRank, etc.)
- Skills and achievements tracking

### Security
- Role-based access control
- Verifier authorization for badge minting
- LocalStorage session management (demo only)

### Known Issues
- Files stored in localStorage (not suitable for production)
- No backend API (client-side only)
- Passwords not hashed (demo only)

## [Unreleased]

### Planned
- Backend API integration
- IPFS document storage
- Email verification
- Password hashing
- Public profile URLs
- Badge verification by employers
